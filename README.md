This repository contains my **Week 5 Python assignment** for the **Power Learn Project (PLP)**.
The focus of this assignment was to practice and demonstrate **Object-Oriented Programming (OOP)** concepts in Python such as:

* Classes and Objects
* Constructors (`__init__`)
* Encapsulation
* Inheritance
* Polymorphism
The project includes several Python programs that illustrate key OOP ideas:
1. House Class Example 
   Demonstrates constructors, encapsulation, and inheritance (with a `Mansion` subclass).

2. Vehicle Polymorphism Example 
   Shows how different vehicle classes implement the same `move()` method in unique ways.

3. Additional OOP Practice
   Includes other examples and comments explaining each concept clearly.
